# Crafatar Mosaic
Simple Python program that uses Crafatar to make a grid of player heads.

1. Provide an input `names.txt` file in the following format:

    ```
    Grian
    Notch
    stampylongnose
    Sethbling
    jeb_
    DanTDM
    MumboJumbo
    Grumm
    AntVenom
    ```

1. Run `python src/main.py` and find the mosaic in `output/mosaic@xxx.png`.
